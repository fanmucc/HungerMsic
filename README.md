## 凡木播放器

### 播放器功能介绍
- 音乐为随机播放
- 将音乐分类
- 点击分类，则播放相应分类中的音乐

### 静态页面问题

 - 页面随着窗口的缩放，页面中的元素也跟着缩放
 - 使用```vh``` 来作为元素宽高的单位
 - 字体图标的引入
 - 背景图片的模糊虚化效果
